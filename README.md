# Cofrinho Pro - Dashboard de Moedas

Sistema de gerenciamento de economias que permite registrar depósitos e retiradas em diferentes moedas estrangeiras, com visualização de gráficos em tempo real e histórico detalhado.

## Funcionalidades
* **Gestão de Saldo**: Adicione ou remova valores de 8 moedas diferentes.
* **Histórico em Tempo Real**: Registro automático de cada transação com data e hora.
* **Gráficos Dinâmicos**: Visualização da distribuição do patrimônio via Chart.js.
* **Segurança**: Sistema protegido com tela de login.

## Tecnologias
* **Backend**: Java 17 + Spring Boot
* **Banco de Dados**: H2 (Em memória)
* **Frontend**: HTML5, CSS3, JavaScript e Chart.js

## Como executar
1. Clone o repositório.
2. Abra no IntelliJ IDEA.
3. Execute a classe `CofrinhoApiApplication`.
4. Acesse `http://localhost:8080` no navegador.
